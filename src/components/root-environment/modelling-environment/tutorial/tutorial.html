Tutorial

On the left side of the screen you can find an image of the Petri net you selected. On the right you can find the modeller, wherein you can model a coverability graph for the Petri net.

You can do almost anything just by using the mouse. By left-clicking on an element of the graph (states and edges) you can select an element. Right-clicking on the modeller will present you a context menu giving you context-dependent options.

To add a state to the graph, press a. You can add an edge between two states by selecting a state, holding the Ctrl button and then selecting another state.

To mark a state as the initial state, select the state and press i. To remove the initial state, select nothing and press i again.

To edit an element of the graph, select the element you want to edit and press e.

You can undo and redo changes using Ctrl+z and Ctrl-y. Alternatively, so you can use Ctrl+Shift+z to undo changes.

To show this tutorial again, press h.
